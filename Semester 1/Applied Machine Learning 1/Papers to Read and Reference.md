## Tips for Citation Management

1. **Use Zotero or Mendeley**: Free reference managers
    
    - Auto-generate bibliographies
    - Organize papers by topic
    - Share with collaborators
2. **Download PDFs immediately**: Save to organized folders
    
    ```
    Papers/
    ├── SVM/
    ├── CNN/
    ├── Augmentation/
    └── Breast_Cancer/
    ```
    
3. **Take notes while reading**: Create summary document
    
    - Key contribution
    - Relevant to your work how?
    - Important figures/tables
    - Quotable sentences
4. **Create citation templates early**: Prepare BibTeX file from day 1

## Red Flags to Avoid

❌ **Don't**:

- Cite papers you haven't read
- Copy citations from other papers without verification
- Use only old papers (pre-2015)
- Ignore the MedMNIST paper
- Over-cite one author/group

✅ **Do**:

- Mix old foundational + recent papers
- Cite primary sources
- Include recent reviews (2020+)
- Verify all citations
- Cite datasets you use
